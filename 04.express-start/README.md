nodejs调试  
-------------------------------------
nodejs 使用 node-inspector 调试  
1. 安装node-inspector  
    npm install -g node-inspector  
2. 启动node-inspector  
    node-inspector  
3. 以debug模式运行node.js应用  
    node --debug app.js  
4. 在chrome中打开URL地址  
    http://127.0.0.1:8080/?ws=127.0.0.1:8080&port=5858
