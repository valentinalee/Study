gitolite安装

1.下载代码
cd $HOME
git clone git://github.com/sitaramc/gitolite gitolite-source
cd gitolite-source
sudo mkdir -p /usr/local/share/gitolite/conf /usr/local/share/gitolite/hooks
sudo src/gl-system-install /usr/local/bin /usr/local/share/gitolite/conf /usr/local/share/gitolite/hooks

2.在客户机上生成ssh key
ssh-keygen -t rsa

3.拷贝客户机的key至服务端
scp /home//.ssh/id_rsa.pub username@server.example.org:/tmp/user.pub

4.创建git用户，并登录
sudo adduser git
su - git

5.安装gitolite
gl-setup /tmp/user.pub

6.从服务端下载 gitolite-admin库
git clone gitolite@server.example.org:gitolite-admin

注：让git / gitolite 知道使用哪一个private key
$ vim ~/.ssh/config
host gitolite
     user git
     hostname yourgitserver.com
     identityfile ~/.ssh/mypubkey

host github.com
	 user git
	 hostname github.com
	 identityfile ~/.ssh/id_rsa

如生成key时未加密码，则使用git gui时需要修改权限
cd ~/.ssh
chmod 700 id_rsa
